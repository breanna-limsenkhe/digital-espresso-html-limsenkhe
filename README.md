# digital-espresso-html-limsenkhe
## architectural strategy 

The Kape Pa Tayo website consists of 4 major HTML pages: `index.html`, `about.html`, `products.html`, and `branches.html`. Each page is linked through a global navigation bar implemented using a `<nav>` element with an unordered list '<ul>' of links, ensuring that users can move seamlessly and smoothly between pages. All images, including menu items and promotional visuals, are stored in a dedicated `images` folder within the project directory. This simple folder structure is enough to keep the files clean and organized, especially with minimal pictures.
